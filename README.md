# VLibras App

Aplicativo para tradução de português para Libras usando o VLibras, com categorias temáticas e sugestão de frases por IA.

## Como usar

1. Coloque as imagens nas respectivas pastas em `img/`
2. Abra o `index.html` no navegador ou publique no GitHub Pages

### Estrutura de imagens

```
img/
├── banners/
│   ├── cumprimentos.png
│   ├── objetos.png
│   └── sentimentos.png
├── cumprimentos/
│   ├── ola.png
│   ├── oi.png
│   ├── bom_dia.png
│   ├── boa_tarde.png
│   └── boa_noite.png
├── objetos/
│   ├── cadeira.png
│   ├── mesa.png
│   ├── quadro.png
│   ├── lapis.png
│   └── mochila.png
└── sentimentos/
    ├── feliz.png
    ├── triste.png
    ├── surpreso.png
    ├── bravo.png
    └── pensativo.png
```

### Publicar no GitHub Pages

1. Crie um repositório no GitHub
2. Envie os arquivos para a branch `main`
3. Vá em **Settings > Pages** e selecione a branch `main` / pasta `root`
4. O app estará disponível em `https://<usuario>.github.io/<repositorio>/`

## Tecnologias

- [VLibras](https://www.gov.br/governodigital/pt-br/vlibras)
- Google Gemini API
- GitHub Pages
